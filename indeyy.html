心得與反思

這門網頁設計課程讓我學到 HTML 跟 CSS 的基本操作，也知道怎麼規劃網頁版面。做專案的時候，有些排版或連結會出問題，但我透過查資料和試不同方法慢慢解決，過程雖然有點挫折，但很有成就感。現在做網頁感覺更順手，也對設計網頁更有興趣了。

課程中遇到的問題及解決方法

問題：CSS 排版常常亂掉，元素不會對齊，整個頁面看起來很亂。
解決方法：我先試了好幾種方法都不行，後來上網查教學，學會之後排版就順多了，也懂得怎麼控制元素的對齊跟間距。
問題：做連結時，點了卻跳不到指定頁面，網頁顯示「找不到檔案」。
解決方法：我仔細檢查了檔案名稱跟路徑，發現大小寫或資料夾位置錯了。改正之後，連結就可以正常跳轉，也學到要注意路徑的正確性。
問題：圖片經常載不出來，或者顯示破圖。
解決方法：我檢查了圖片存放的資料夾、檔名，還有 HTML 裡使用的相對路徑。修正後圖片就正常顯示了，也知道了管理檔案路徑的重要性。
問題：剛開始學 HTML 時，不知道標籤要怎麼搭配，內容排版亂七八糟。
解決方法：我多看了網路範例，也慢慢試不同標籤搭配，慢慢就懂了結構和語法，網頁看起來整齊很多，也對 HTML 有更多自信。
